## Installation & Usage

### 1. Clone the Repository

#### macOS/Linux
```bash
sudo git clone https://github.com/xircons/store.atelien.git
```
#### Windows
```bash 
git clone https://github.com/xircons/store.atelien.git
```

### 2.  Setup Backend
```bash 
cd server && npm start
```
